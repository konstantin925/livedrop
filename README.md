# LiveDrop

Mobile-first deals app for local drops, online deals, claims, catalog saves, and admin publishing.

## Run locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Configure your local environment variables in `.env.local`
3. Run the app:
   `npm run dev`

## Build

`npm run build`
